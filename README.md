# Titanic Survival Prediction

## Цель проекта

В данном проекте решается задача Titanic Survival Prediction с использованием DVC + MLFlow.

## Как запустить

!!! в dvc.yaml укажите нужный python интерпретатор. Я работал на Mac OS, поэтому указал его.

```bash
# 1. Установите необходимые библиотеки
pip install -r requirements.txt

# 2. Проведите подготовку данных
dvc repro prepare

# 3. Обучите модель
dvc repro train
```

## Краткое описание пайплайна

1. **prepare** — этап подготовки данных: очистка, преобразование признаков, деление на train/test выборки.
2. **train** — этап обучения модели Random Forest на подготовленных данных с сохранением метрик и артефактов с помощью MLflow.
3. Все параметры пайплайна конфигурируются через файл `params.yaml`.
4. Для воспроизводимости и отслеживания артефактов используется DVC.
